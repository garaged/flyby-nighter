# SPEC-0023: Score System

Status: Proposed  
Milestone: M4

## Goal

Make scoring more interesting while keeping it understandable and deterministic.

## Scope

- Score sources.
- Optional combo or streak logic.
- Route completion bonus.
- Remaining HP bonus.
- Clear score feedback.

## Non-goals

- No online leaderboard.
- No economy.
- No permanent upgrade purchases.

## Acceptance criteria

1. Score sources are documented.
2. Score changes are visible during play.
3. Any combo or streak rule is deterministic and test-covered.
4. Completion and remaining HP bonuses are clear.
5. `swift test` passes.
