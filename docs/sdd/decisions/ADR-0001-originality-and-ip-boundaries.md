# ADR-0001: Originality and IP Boundaries

Status: Accepted  
Date: 2026-07-01

## Context

Flyby Nighter is inspired by the broad arcade shooter genre, not by a license to recreate any specific legacy game. Classic arcade games may still have active copyright and trademark protection.

## Decision

Flyby Nighter will be an original game with original expression.

The project may use broad, unprotectable genre ideas such as:

- Arcade shooting.
- Scrolling routes.
- Dodging obstacles.
- Power pickups.
- Score chasing.
- Enemy waves.
- Boss or exit encounters in future milestones.

The project must not copy or recreate protected expression from Vanguard or any other specific game, including:

- Code or ROM behavior.
- Sprites, tiles, ships, enemies, bosses, maps, or cabinet/flyer/manual art.
- Music, sound effects, voice samples, or sound-alike sequences.
- Distinctive level layouts, exact pacing, zone order, names, lore, UI arrangement, or marketing identity.
- Product naming that implies an official remake, sequel, port, or endorsement.

## Consequences

- All names, map layouts, enemy identities, pickups, art, audio, and UI must be original.
- Specs should describe intent and mechanics, not clone references.
- Public documentation should call the game "retro-inspired" or "arcade-inspired," not a remake or clone.
- Any temporary placeholder assets must be clearly original or generated by the project.
