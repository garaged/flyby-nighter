# Flyby Nighter

An original retro-inspired arcade shooter built around fast tunnel navigation, directional shooting, score chasing, and handcrafted sci-fi hazards.

## Development workflow

This project uses Spec Driven Development.

Start here:

- `AGENTS.md` for AI/Codex implementation rules.
- `docs/sdd/README.md` for the SDD workflow.
- `docs/sdd/registry.md` for spec status.
- `docs/sdd/milestones/M0-minimal-playable.md` for the first playable target.
- `docs/sdd/questions.md` for active design decisions.

## Current target

M0: one playable map with player flight, shooting, a few attacking enemies, obstacles, collectible gifts, powers, score, win/fail state, and restart.
